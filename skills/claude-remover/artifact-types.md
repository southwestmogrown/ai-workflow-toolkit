# AI Tool Artifact Catalogue

A comprehensive list of files, patterns, and artifacts left behind by AI coding
tools, organized by tool. Use this as a lookup when scanning an unfamiliar project.

---

## Claude (Anthropic)

### Files
| File/Pattern | Description | Action |
|---|---|---|
| `CLAUDE.md` | Root-level Claude instruction file | Delete |
| `**/CLAUDE.md` | Nested Claude instruction files | Delete (unless in `.github/`) |
| `*.claude.md` | Custom Claude context files | Delete |
| `.claude/` | Claude project config directory | Delete |
| `.claude_context` | Claude context dump | Delete |

### Code patterns
| Pattern | Description | Action |
|---|---|---|
| `"claude-(opus\|sonnet\|haiku)-[0-9]"` | Hardcoded model string | Replace with env var |
| `ANTHROPIC_API_KEY = "sk-ant-..."` | Hardcoded API key | Delete + rotate key |
| `// Generated by Claude` | Attribution comment | Delete or trim |
| `# Claude: ` | Inline Claude note | Delete |

## Generic AI patterns (any tool)

### Comment patterns to flag or trim
```
// TODO: [AI|Claude|Copilot|GPT] — ...
// NOTE: Claude suggested ...
// AI: this might need to be ...
// Generated by [AI tool]
// This function was written by Claude
// [AI-generated] ...
/* Claude: consider refactoring ... */
```

### Debug/dev prompt patterns to flag
```typescript
const systemPrompt = "You are a helpful assistant..."  // hardcoded
const testPrompt = "..."                               // test artifact
const devPrompt = "..."                                // dev artifact
const debugMessage = "..."                             // debug artifact
```

### Oversized comment blocks
Any comment block exceeding ~8 lines that:
- Explains what obvious code does (not why)
- Contains phrases like "this function", "this component", "the above code"
- Reads like a tutorial rather than a comment
- Has identical structure to typical AI explanations

These are almost always AI-generated explanations that add noise without value.

## Files that look like AI artifacts but are NOT

These should be left alone:

| File | Why it's safe | Notes |
|---|---|---|
| `.github/CLAUDE.md` | Part of repo docs | Only delete root-level `CLAUDE.md` |
| `src/lib/anthropic.ts` | Runtime SDK client | That's the product |
| `src/prompts/system.ts` | Production prompt config | That's the product |
| `types/anthropic.d.ts` | Type definitions | Runtime dependency |
| `__tests__/ai.test.ts` | Tests for AI features | Keep |
| `docs/ai-features.md` | User-facing docs | Keep |
| `.env.example` | Example env file | Keep (but verify no real keys) |
